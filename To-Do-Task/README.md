# Simple Task Saver using Python

This is a basic Python project that stores tasks in a text file.

The program:
- Takes task input from the user
- Saves the task into a file
- Marks tasks as completed
- Clears the task file when completed

This project was created for practicing:
- Python file handling
- Functions
- Conditions
- Exception handling
- Basic task management logic

---

# Features

- Add tasks
- Save tasks in `savingtask.txt`
- Handle file errors
- Mark tasks as completed
- Simple beginner-friendly logic

---

# Technologies Used

- Python
- Text File Handling
- OS Module

---

1. Install Python
2. Save the code as `task.py`
3. Run the program
